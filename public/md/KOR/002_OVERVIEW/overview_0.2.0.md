# NewLearnNote 개요

## 핵심 기능

### AI 어시스턴스 (Premium)
VSCode Copilot처럼 에디터 안에서 Gemini AI와 대화하며 노트 작업을 진행합니다.

- **문서 작업 자동화**: "이 PDF 내용을 마크다운으로 정리해줘", "이 노트를 요약해줘"
- **학습 지원**: "이 개념을 초등학생도 이해할 수 있게 설명해줘", "예제 코드 추가해줘"
- **구조 개선**: "이 글의 논리 구조를 개선해줘", "목차를 생성해줘"
- **참고자료 통합**: 여러 파일을 참조하여 하나의 정리된 노트로 재구성

### 지식 네트워크
다른 사용자의 검증된 노트를 참조하며 함께 성장하는 집단지성 생태계입니다.

- **맥락 기반 추천**: 현재 학습 중인 주제와 관련된 노트 자동 추천
- **살아있는 참조**: 다른 사용자 노트를 임베드하면 원본 업데이트가 자동 반영
- **신뢰 시스템**: 전문 분야별 평판과 커뮤니티 검증으로 품질 보장

### 완벽한 동기화
데스크톱 앱의 강력한 로컬 파일 관리와 클라우드의 편리함을 결합했습니다.

- **Electron 데스크톱 앱**: 로컬 파일 시스템 완벽 통합, 오프라인 지원
- **웹 인터페이스**: 어디서든 브라우저로 즉시 접근
- **충돌 해결**: 다중 기기 간 자동 동기화 및 버전 관리

## 사용 시나리오

### 대학생의 하루
강의 녹음 파일과 교재 PDF를 열어두고 AI에게 "이 두 자료를 참고해서 시험 대비 요약 노트 만들어줘"라고 요청합니다. 다른 학생이 작성한 같은 과목 정리 노트가 추천되어 참고하며 이해를 넓힙니다.

### 개발자의 학습
공식 문서와 블로그 글을 읽다가 "이 내용을 내 프로젝트에 적용하는 방법을 단계별로 정리해줘"라고 AI에게 요청합니다. 같은 기술을 사용한 다른 개발자의 노트가 자동으로 제안되어 참고합니다.

### 연구자의 논문 관리
"이 세 편의 논문에서 공통된 연구 방법론을 추출해서 표로 정리해줘"라고 AI에게 요청합니다. 같은 분야 연구자들이 공유한 논문 요약을 발견하고, 참고문헌을 자동으로 관리합니다.
